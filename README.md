pyspectator
===========

Cross-platform tool for monitoring resources of OS with web-interface.

How to install
===========

Run as root user:
```bash
pip install pyspectator
```

Requirements
===========

- Python version: 3.X
- Packages: psutil, netifaces, wmi (only on Windows), enum34 (only on python 3.0.0 - 3.4.0)
